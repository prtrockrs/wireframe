## 3d wireframe model viewer

- takes in vertices/edges of a obj file and renders it in wireframe
- rotates slowly to show full model
- up/down arrow keys to zoom in/out

## How to Run

Do 

`make`

then

`./wireframe <filename>`

where filename for `testmodel.obj` would be `testmodel`.

All .obj files are expected to be in the `objects` folder
